# Online Banking System (Java)

This is a simple console-based banking system built using core Java.  
The project allows users to perform basic banking operations such as creating accounts, depositing and withdrawing money, checking balances, and transferring funds between accounts.

## Features
- Create a new bank account
- Deposit money into an account
- Withdraw money from an account
- Check current account balance
- Transfer funds between two accounts
- Data is stored using file serialisationm

## Technologies Used
- Java (JDK 8 or higher)
- File handling for data persistence
- Object-Oriented Programming concepts
